Project Point: It started as a simple project. A mood lamp.

Project Point: It is my own server implementation with my own mini PC in my attic.

The backend running on Node express JS, using some third party components for it too. Currently not using HTTP for it but i will in the future. The general idea is that i wanted my own Drive beseides google or any other party. I reforged an old MINI PC and i writed a little little NODE js server on it. The project growed to my group of friends. There are currently 6 people who can use it. The emails are prewrited to the server side and only the members of the Koma'k group can register on the web and login. They all have a dynamically created account (when they registered) with MYSQL database, their own drive folders on the PC. Everyone able to use the "public" drive they can upload pictures and videos to it freely as well as to their accounts which they can see only. On the main page there is an Event alert, everyone is able to create an event on their account menu wich is displayed on the front page when it is created. The created event is visible to anyone who is logged in. Everyone is able to set a nickname for itself, and a profile photo too. This is required when someone wants to use the chat feature of the web wich is implemented with socket.io.
